Welcome to Pi Reel's Git Hub Page.

Mozilla Club Omaha is excited to be contributing the "Pi Reel Science and Engineering Animator Project" for Mozsprint 2017.

Special thanks to the following Open Source Projects for making this alpha build of Pi Reel happen.

Mozilla Foundation || Mozilla Learning Network || Mozilla Science || Cloud 9 || x-tag || Apache || XAMMP

Pi Reel, is a simple animation editor tailor made for science and engineering. The simple user interface gives you a basic platform to customize however you need the animator to be. It can accomplish this a couple of different ways, but the featured way we hope to roll out at this year's Mozsprint are the "Pi Reel Key Definition" and "Pi Reel Frame Definition" consoles.

Below is a list of some of the products of Pi Reel and the features Pi Reel will be bringing to demo and collaborate on at Mozsprint 2017.

  1. Pi Reel Blogging Engine
    1. Open Science Blogging
    2. Open Engineering Blogging
    3. Science Fiction Story Blogging
    4. Graphic Novel Blogging
  2. Pi Reel Open Workflow Engine
    1. Issues Tracker Web App
    2. Roadmap Web App
    3. Guide Doc Viewer
  3. Pi Reel Science and Engineering Animator
    1. Key Definition Console
    2. Frame Definition Console
    3. Canvas Tray
    4. Media Feed
